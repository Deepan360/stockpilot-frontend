# StockPilot AI - Frontend

StockPilot AI is a modern inventory intelligence platform built using React and Material UI.

## Features

* Secure Login & Registration
* Dashboard Analytics
* Product Management
* Low Stock Monitoring
* Inventory Health Metrics
* Responsive Design
* Analytics Dashboard
* User Profile Management

## Tech Stack

* React.js
* Vite
* Material UI
* Axios
* React Router
* React Hot Toast

## Environment Variables

Create a `.env` file:

VITE_API_URL=your_backend_api_url

## Installation

npm install

npm run dev

## Build

npm run build

## Pages

* Dashboard
* Inventory
* Stock Alerts
* Analytics
* Profile
